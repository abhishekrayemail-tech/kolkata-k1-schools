# Kolkata LKG/K1 Admissions Directory 2026–27

A comprehensive, scored directory of 72 schools in Kolkata for Pre-Nursery / Nursery / LKG admissions — with a focus on **SEN (Special Educational Needs) friendliness** and **academic performance**.

## Features

- **72 schools** scored on SEN Friendliness (1–5) and Academic Performance (1–5)
- **Evidence-based SEN scoring** with proof links (CIS SEN Dept page, Heritage 12 Special Educators, etc.)
- **Admission window tracker** with dates, deadlines, and next-year planning reminders
- **Personal notes tracker** — record your discussions, status, and follow-ups per school
- **Firebase sync** — notes persist across phone, laptop, tablet in real-time
- **Class-level filter** — filter by Pre-Nursery, Nursery, LKG, UKG, or Class 1
- **Pagination, sorting, search** across all schools
- **Expandable detail rows** — click "Details" to see full info, admission window, SEN evidence, and your notes
- **Export/Import** notes as TXT or JSON
